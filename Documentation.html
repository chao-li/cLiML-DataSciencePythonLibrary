What is this?¶

This package contains a collection of functions I wrote as I was working on my Data Science projects. They perform some tedious and repetitive operations that are necessary when making Machine Learning models. These functions builds on top of popular libraries such as: numpy, pandas, matplot.lib, sklearn.

Some functionalities:¶

  • Find all outliers within a dataframe.
  • Perform EDA on numerical variables
  • Perform EDA on categorical variables
  • Convert all of your categorical variables into dummy variables
  • Perform feature selection using RFECV, SelectKBest, Lasso all at the same time, output results for comparison.

cLiML.eda.findOutlier¶

This method will find all the outliers of the numerical variables within a DataFrame. Don't worry about any columns that are of Object type. This method will automatically ignore those columns.

input:¶

  • DataFrame: Takes a dataframe.

cLiML.eda.edaCategorical¶

This method will plot each categorical variables against the target variable as a bar graph. Note that the target variable is aggregated using either a 'median' or 'mean' method. Effectivly what we are doing is df.groupby(category).target.mean() or df.groupby(category).target.median()

Note, this will be performed for every column, regardless if they are Object or Numbers. So if you don't want to analyse your number variables as a categorical variable, make sure to exclude those columns.

Plots will be presented in two columns of subplots.

input:¶

  • df: The dataframe you want to analyze.
  • target: The name of your target variable, note this must be a string.
  • aggre = 'median': The default aggregate function is median, type 'mean' if you want mean.
  • max_y_axis = None: This is the maximum y axis value you want. If left blank, it will automatically use the maximum value of your target variable.

Example¶

In [5]:
# Import the method, remember to also include matplotlib inline
from cLiML import eda
%config InlineBackend.figure_format = 'retina'
%matplotlib inline

# Note that the aggregate function is not selected, so median is used by default
# 
eda.edaCategorical(df[['MSSubClass', 'Street', 'SalePrice','MoSold']], target = 'SalePrice', max_y_axis = 300000)

cLiML.eda.edaContinuous¶

This method will plot each continuous variables against the target variable as a scatter plot.

Note, this method will automatically recognize numeric columns. Object columns are automatically ignored.

Plots will be presented in two columns of subplots.

input:¶

  • df: The dataframe you want to analyze.
  • target: The name of your target variable, note this must be a string.
  • max_y_axis = None: This is the maximum y axis value you want. If left blank, it will automatically use the maximum value of your target variable.

Example¶

In [10]:
# Import the method, remember to also include matplotlib inline
from cLiML import eda
%config InlineBackend.figure_format = 'retina'
%matplotlib inline


# 
eda.edaContinuous(df[['Street', 'SalePrice', 'LotFrontage','LotArea', 'YrSold', 'MoSold']], target = 'SalePrice')

cLiML.model_prep.dummify¶

This method will dummify all the Object type variables in your dataframe. You can also specify exactly which columns you want to get dummified. Note if you want a numerical column to be dummmified, you must specify it.

input:¶

  • df: Your dataframe
  • categorical_var = None: By default, all Object type columns will be dummified. Or you can input a list of column names that you want to get dummified.

Example¶

In [12]:
# Note only the street type is dummified, as it is the only Object type column
from cLiML import model_prep
model_prep.dummify(df[['MSSubClass', 'Street', 'SalePrice','MoSold']]).head()

cLiML.model_prep.feature_select_linear_reg¶

This will perform RFECV, SelectKBest, Lasso feature selection on your dataframe, assuming you are going to perform a linear regression.

The results of all 3 feature selections are outputted side by side for comparison.

input:¶

  • X: Takes a 2D dataframe.
  • y: Takes a 1D series.
  • cv: The number of cross validation fold you want to perform
